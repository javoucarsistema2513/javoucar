# JávouCar - Sistema de Alertas Veiculares

Este é o repositório do projeto JávouCar, um sistema inteligente para gerenciamento de alertas veiculares e comunicação entre motoristas.

## Sobre o Projeto

JávouCar é uma aplicação web que permite aos motoristas enviar e receber alertas em tempo real sobre situações específicas relacionadas a veículos, como bloqueios, faróis acesos, pneus murchos, entre outros.

## Funcionalidades

- Sistema de autenticação de usuários
- Cadastro de veículos
- Envio de alertas pré-definidos ou personalizados
- Recebimento de alertas em tempo real
- Interface responsiva para dispositivos móveis

## Tecnologias Utilizadas

- React com TypeScript
- Vite como bundler
- Supabase para backend e autenticação
- Tailwind CSS para estilização
- Lucide React para ícones

## Como Executar Localmente

**Pré-requisitos:** Node.js

1. Instale as dependências:
   `npm install`
2. Configure as variáveis de ambiente no arquivo `.env.local`
3. Execute a aplicação:
   `npm run dev`

## Deploy

Para fazer o deploy da aplicação, você pode utilizar serviços como Vercel, Netlify ou Render.
